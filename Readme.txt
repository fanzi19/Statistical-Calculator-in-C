# Statistical Calculator

This C program reads numerical data from a file, calculates various statistical measures, and provides insightful results. It includes functions for computing the mean, mode, geometric mean, harmonic mean, median, and standard deviation of a dataset.

## Table of Contents
- [Features](#features)
- [Getting Started](#getting-started)
  - [Prerequisites](#prerequisites)
  - [Usage](#usage)


## Features

- Read Data: Read numerical data from a file.
- Calculate Mean: Compute the mean of the dataset.
- Calculate Mode: Determine the mode(s) of the dataset.
- Calculate Geometric Mean: Compute the geometric mean of the dataset.
- Calculate Harmonic Mean: Compute the harmonic mean of the dataset.
- Calculate Median: Determine the median of the dataset.
- Calculate Standard Deviation: Compute the standard deviation of the dataset.

## Getting Started

### Prerequisites

Ensure you have a C compiler installed on your system.

### Usage

Clone the repository:

   ```bash
   git clone https://github.com/yourusername/statistical-calculator.git

Compile the program:
gcc -o stats_calculator stats_calculator.c -lm

Run the program:
./stats_calculator your_data_file.txt

Replace your_data_file.txt with the path to your data file.
